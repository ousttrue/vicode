# vicode
vscode style vi.

## layout

```
+-------+------+
|sidebar|editor|
|       +------+
|       |panel |
+-------+------+
|command       |
+--------------+
```

## keyboard shortcut

- `c-w h`: focus sidebar
- `c-w j`: forus panel
- `c-w k`: focus editor
- `c-w l`: focus editor
- `m-h`: focus prev tab
- `m-l`: focus next tab
